# Deploying Docker in AWS

## Setup:

### Setup your credentials

Create an access token for eb and add it to your local configuration.

```
# ~/.aws-config
[default]
aws_access_key_id=<id>
aws_secret_access_key=<access-key>
region=eu-central-1
```
